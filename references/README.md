# References Directory

This directory contains reference materials, documentation, and research papers.

## Contents

- Research papers and articles
- Data dictionaries and codebooks
- API documentation
- Project specifications and requirements
- Literature reviews
- Technical documentation

## Guidelines

1. Organize by topic or category
2. Use descriptive filenames
3. Include citations and sources
4. Keep PDFs of important papers
5. Document data schemas and field definitions
6. Maintain a bibliography file

## Suggested Files

- `data_dictionary.md` - Description of all data fields
- `api_documentation.md` - API endpoints and usage
- `literature_review.md` - Summary of relevant research
- `methodology.md` - Project methodology and approach
